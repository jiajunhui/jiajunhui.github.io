C++实现二分查找

发表于 2017-10-17 分类于 C++

二分查找又称折半查找，二分查找是针对有序数组所用的一种快速查找元素的方法。

二分查找的条件及优缺点

条件：针对有序数组（元素从小到大或从大到小）
优点：查询速度较快，时间复杂度为O（n）
缺点：有硬性条件的限制，而且即使查到后，插入与删除困难。
