Kotlin基础

发表于 2019-07-02 分类于 Kotlin

Kotlin（科特林）是一个用于现代多平台应用的静态编程语言，由 JetBrains 开发。
Kotlin可以编译成Java字节码，也可以编译成JavaScript，方便在没有JVM的设备上运行。除此之外Kotlin还可以编译成二进制代码直接运行在机器上（例如嵌入式设备或 iOS）。
Kotlin已正式成为Android官方支持开发语言。
