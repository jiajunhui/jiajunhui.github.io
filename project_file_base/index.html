文件操作工具库FileBase

发表于 2017-03-11 | 分类于 我的开源项目 |

介绍

文件的操作在项目中是必然用到的，为了方便这些操作，封装了个库FileBase.
包含了目录管理基类、对文件的常用操作、对Assets的操作、存储器信息的获取以及一些其他的操作。

主要功能

目录管理助手

我们开发应用时一般都会有这样的需求场景：在存储器里面为自己的APP创建一个存储目录，用于存放下载的文件、本地配置信息、缓存的文件、临时文件等等。我们习惯的做法是：在存储器下判断是否存在我们的文件夹A，如果没有，创建一个文件夹A，然后在文件夹A里面再分别创建cache、download、temp等目录。如果再有别的主目录需要管理的话，看上去就相当混乱。那么能不能简化这些操作，而且让代码看上去条理清晰呢。我的方案来了，封装了一个基类，使用时只需要继承并提供一个主目录名称，需要获取某个子目录的路径时，直接提供文件夹的名字即可，无需关心文件夹在不在，需不需要创建等问题。
